面試 - 鼎羿

Posted on 2020-03-03 | In interview

公司名稱

鼎羿科技有限公司

地址

台北市內湖區瑞光路 298 號 8 樓

應徵職位

前端工程師

面試日期

2020/03/03 10:00

攜帶

帶履歷表、筆電、作品

面試流程

先由櫃台帶到會議室，並給了一份專業技能試卷 20 分鐘作答。

專業技能考題大概如下

let 與 var
(1)

for (var i = 1; i <= 5; i++) {
  setTimeout(() => {
    console.log(i);
  }, 0);
}
console.log(i);

// 6
// 6
// 6
// 6
// 6
// 6

(2)

for (let i = 1; i <= 5; i++) {
  setTimeout(() => {
    console.log(i);
  }, 0);
}
console.log(i);

// i is not defined
// 1
// 2
// 3
// 4
// 5
請寫出三種讓子元素於父元素裡置中的方法

Ans:

(1) flex

  .parent{
    display:flex;
    justify-content:center;
  }

   .child{}

(2) position

.parent{
  position: relative;
}

.child{
  position: absolute;
  top: 0;
  left: 50%;
  transform: translateX(-50%);
}

(3) margin

.parent{
}

.child{
  margin: 0 auto;
}
用 async await 寫出延遲一秒輸出 console.log(1)

Ans:

function delay(t) {
  return new Promise((resolve, reject) => {
    setTimeout(() => {
      resolve();
    }, t);
  });
}
(async function() {
  await delay(1000);
  console.log('1');
})();
ary=[1,2,3,…N] 使用 ary 方法算出總和

(1) reduce

let ary = [1, 2, 3, 4];
let total = ary.reduce((prev, next) => prev + next, 0);
console.log(total);

(2) foreach

let ary = [1, 2, 3, 4];
let total = 0;
ary.forEach(item => (total += item));
console.log(total);
還有一些題目忘記了

接著與技術長面談

根據剛才的試卷作答探討，也請我自我介紹，展示自己的作品並做簡單的介紹，技術長也分享了自己的一些觀點。

再來與 HR 面談

HR 介紹了一下公司及環境，並了解為何離開前一間公司。

小結

總時長約花 1.5 小時，是間博弈公司，公司強調工作環境的氣氛，上班期間都有音樂聽，大家也都會訂便當一起吃，也有懶骨頭中午可以睡，公司環境、氣氛不錯，與面試官也都聊得很開心。

最後 offer get，也接受，至於後面的工作狀況又是另一個故事了。
