專業技能考題大概如下

let, var 與 settimeout (選擇題)
// let 與 var 在if,else,for,while會有差 let,const => block scope ; var => function scope

function b() {
  for (var i = 1; i <= 3; i++) {
    setTimeout(() => {
      console.log("settimeout裡面", i);
    }, 0);
    console.log("settimeout外面", i);
  }
}
function c() {
  for (let i = 1; i <= 3; i++) {
    setTimeout(() => {
      console.log(i);
    }, 0);
  }
}
b(); // 內4,4,4 外1,2,3
c(); // 1,2,3
obj {} (選擇題)
function c(obj) {
  if (obj === { name: "Howie" }) {
    console.log("a");
  } else if (obj == { name: "Howie" }) {
    console.log("b");
  } else {
    console.log("c");
  }
}
c({ name: "Howie" }); // undefined
settimeout 非同步會放後面執行 (選擇題)
function a() {
  console.log("first");
}
function b() {
  setTimeout(() => {
    console.log("second");
  }, 0);
}
function c() {
  console.log("third");
}
b();
a();
c();
// first
// third
// second

簡述 SPA
Vue 父子如何傳值

Ans:

父 -> 子
(1) 子元件在 props 中建立一個屬性，用以接收父元件傳過來的值
(2) 父元件中註冊子元件
(3) 在子元件標籤中新增子元件 props 中建立的屬性
(4) 把需要傳給子元件的值賦給該屬性

子 -> 父
(1) 子元件中需要以某種方式例如點選事件的方法來觸發一個自定義事件
(2) 將需要傳的值作為$emit 的第二個引數，該值將作為實參傳給響應自定義事件的方法
(3) 在父元件中註冊子元件並在子元件標籤上繫結對自定義事件的監聽

簡述 Vue 生命週期

Ans:

beforeCreate：初始化
created: 建立完成。資料 $data 已可取得，但 $el 屬性尚未被建立。
beforeMount：執行元素掛載之前，代表 $el 尚未被建立
mounted：元素已掛載， $el 被建立
beforeUpdate：當資料改變被呼叫使用，還不會渲染 View
updated：當資料更新完成，則驅動 DOM 元素重新渲染畫面 View
beforeDestroy：Vue 還可使用
destroyed：Vue 銷毀，所有的 DOM 元素綁定被解除、移除偵聽事件、Vue child 也被一併銷毀
