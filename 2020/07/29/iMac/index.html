iMac

發表於 2020-07-29 分類於 working

在一些因緣際會下，將電腦從 Windows 換成 iMac，對於從未使用過 Apple 產品的我來說實在是個大挑戰阿!!

好在原先主人的環境大部分都有保留，來簡單做個整理

快捷鍵

關閉視窗/程式 ——> Command + Q or W
切輸入法 —–> 原大小寫位置按鍵 (需裝有其他輸入法)
滑鼠手勢設定 —–> 可開啟右鍵、滾動等各種手勢
重做 —–> Command + Shift + Z (Windows 為 Ctrl + Y)
切換視窗 —–> Command + Tab
重整 —–> Command + R
捲到最上方 —–> Command + 箭頭上
截圖 —–> Shift + 3 (尚未驗證)
放大顯示 —–> Command + + (尚未驗證)
回預設 —–> Command + 0 (尚未驗證)
改檔名 —–> Enter (尚未驗證)
刪除檔案 —–> Command + Delete (尚未驗證)
快速切換桌面 —–> Fn + F11 (尚未驗證)
