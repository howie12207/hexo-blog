閉包

Posted on 2020-04-12 In javascript

何謂閉包?

我自己的解釋會是，一個函數(外部函數)內的局部變量，是無法從外面直接取得，但可以透過在此函數(外部函數)內在建一個函數(內部函數)，而此內部函數可以返回外部函數的值。所以可以透過呼叫內部函數返回外部函數的值，並將此值重複使用。

以下兩變數 myFunc1 與 myFunc2 的值會不同

function makeFunc() {
  let ran = Math.random();
  return ran;
}

let myFunc1 = makeFunc();
let myFunc2 = makeFunc();
console.log(myFunc1);
console.log(myFunc2);

為什麼會這樣?

先了解此代碼的運作流程

1.執行 makeFunc()並賦值給 myFunc1 變數

let myFunc1 = makeFunc();

2.執行 makeFunc()並賦值給 myFunc2 變數

let myFunc2 = makeFunc();

3.4 打印 myFunc1 與 myFunc2 變數

console.log(myFunc1);
console.log(myFunc2);

由此可知因為每次執行 makeFunc()，產生的 Math.randon()值會不同，所以變數 myFunc1 與 myFunc2 值會不同。

那要怎麼讓這兩變數值相等呢?

function makeFunc() {
  let ran = Math.random();
  function insideFunc() {
    return ran;
  }
  return insideFunc;
}

let f = makeFunc();
let myFunc1 = f();
let myFunc2 = f();
console.log(myFunc1);
console.log(myFunc2);

此時兩次變數的值就會相等

來了解一下他的運行方式

1.執行 makeFunc()

let f = makeFunc();

2.makeFunc()就是執行以下代碼

let ran = Math.random();
return insideFunc;

3.執行 f()，並賦值給 myFunc1 變數

let myFunc1 = f();

4.f()就是執行以下代碼

return ran;

5.執行 f()，並賦值給 myFunc2 變數

let myFunc1 = f();

6.f()就是執行以下代碼

return ran;

7.打印 myFunc1 與 myFunc2 變數

console.log(myFunc1);
console.log(myFunc2);

由此可知，Math.random()只被執行一次，所以 randon 值只產生一次
而 f()即是 makeFunc()()，當時 makeFunc()已返回 insideFunc
就變成執行 insideFunc()
而返回值 ran 在內部函數沒有所以在向外部函數查找
