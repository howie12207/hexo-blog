陣列的各種方法

Posted on 2020-04-12 In javascript

常常忘記陣列的操作方法，所以以自己好懂的方法紀錄一下。

Array.prototype.filter() 將符合的元素返回新陣列。

const arr = [1, 30, 39, 29, 10, 13];

let newAry = arr.filter(item => item < 20);
console.log(newAry); // [1, 10, 13]

Array.prototype.reduce() 將一個累加器及陣列每個元素傳入，返回為單一值。

const ary = [1, 3, 5, 7];

let result = ary.reduce((prev, next) => prev + next, 0);

console.log(result); // 16

Array.prototype.every() 是否陣列內所有元素都符合，返回 true or false。

const arr = [1, 30, 39, 29, 10, 13];

let result1 = arr.every(item => item < 50); // true
let result2 = arr.every(item => item < 20); // false

Array.prototype.some() 是否陣列內有任一 元素符合，返回 true or false。

const arr = [1, 30, 39, 29, 10, 13];

let result1 = arr.some(item => item < 50); // true
let result2 = arr.some(item => item < 20); // true

Array.prototype.find() 返回第一個符合的元素。

const arr1 = [1, 30, 39, 29, 10, 13];

let result1 = arr1.find(item => item > 20);
console.log(result1); // 30

const ary2 = [
  { name: "hulk", age: 20 },
  { name: "howie", age: 30 },
];

let result2 = ary2.find((item) => item.age === 20);
console.log(result2); // { name: "hulk", age: 20 }

Array.prototype.includes() 是否陣列內有符合的元素，返回 true or false。

const arr = [1, 30, 39, 29, 10, 13];

let result = arr.includes(30);
console.log(result); // true

Array.prototype.map() 對陣列內每個元素操作，返回一個新陣列。

const ary = [1, 4, 9, 16];

const newAry = ary.map(x => x * 2);

console.log(newAry); // [2, 8, 18, 32]

Array.prototype.forEach() 對陣列內每個元素操作。

const array1 = ['a', 'b', 'c'];

array1.forEach(element => console.log(element)); // a, b, c

Array.prototype.join() 將陣列中的所有元素以某字符或空(預設,)串成字串，返回此字串。

const elements = ['Fire', 'Air', 'Water'];
let result1 = elements.join();
console.log(result1) // Fire,Air,Water
let result2 = elements.join('');
console.log(result2) // FireAirWater
let result3 = elements.join('*-*');
console.log(result3) // Fire*-*Air*-*Water

Array.prototype.splice() 增、刪原陣列內容。

array.splice(start[, deleteCount[, item1[, item2[, …]]]])

start 開始的位置，起始為 0 ，-1 為倒數第一個，只填此處表從此位置(包含)後面全刪
deleteCount 表刪除的數量，0 表示不刪除
item1, item2 表添加的內容

let ary1 = [1, 3, 5];
ary1.splice(1, 0, 2);
console.log(ary1); // [1, 2, 3, 5]

let ary2 = [1, 2, 3, 4, 5]
ary2.splice(2, 2);
console.log(ary2); // [1, 2, 5]

let ary3 = [1, 2, 3, 4, 5]
ary4 = ary3.splice(2);
console.log(ary3); // [1, 2]
console.log(ary4); // [3, 4, 5]

Array.prototype.slice() 將選擇範圍的內容返回新陣列，原陣列內容不變。

arr.slice([begin[, end]])

begin 開始的位置，起始為 0，-1 為倒數第一個，只填此處表從此位置(包含)後面全保留
end 至哪個位置(不包含)

let ary1 = [1, 2, 3, 4, 5];
let newAry1 = ary1.slice(1, 3);
console.log(newAry1); // [2, 3]

let newAry2 = ary1.slice(2);
console.log(newAry2); // [3, 4, 5]

Array.prototype.sort() 將陣列內所有元素排序，覆寫此陣列。

const array1 = [1, 30, 4, 21, 100000];
array1.sort();
console.log(array1); // [1, 100000, 21, 30, 4]

const ary2 = [1, 30, 4, 21, 100000];
ary2.sort((a,b) => a - b);
console.log(ary2); // [1, 4, 21, 30, 100000]

Array.from() 將類陣列或可迭代物件轉成陣列。

console.log(Array.from('foo')); // ['f', 'o', 'o']

console.log(Array.from([1,2,3], x => x + x)); // [2, 4, 6]
