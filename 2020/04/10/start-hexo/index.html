開始使用 Hexo

為什麼開始使用 Blog?

在這學習程式語言的階段，有太多東西需要去學習，而記憶力就只有那麼一點，透過部落格的紀錄方式，方便自己以後回來查找，也可以記錄一些自己的生活。

部落格是怎麼製作的?

爬了一下文，發現大部分的人都是使用’hexo’，所以就找了幾篇關於 hexo 的教學文來看

建置

1. 開啟 VScode，在終端機中使用以下指令，將 hexo 套件建立在全域下。

npm install -g hexo-cli

2. 建立 hexo 專案

hexo init <資料夾名稱>

3. 進入資料夾並安裝

cd <資料夾名稱>
npm install

4. 產生靜態檔案

hexo g

5. 建立 hexo git (到_config.yml 更改配置)

npm install hexo-deployer-git --save

6. 將檔案 push 上去 (如果沒有 SSH 可能會發生問題)

hexo d

啟動 Server

hexo s

新增文章

layout -> 選擇格式
title -> 輸入標題
將新增一個 md 檔在 scaffolds 資料夾內

hexo new [layout] <title>

修改配置檔

_config.yml

#Site

title: Howie's blog // 標題
subtitle: "" // 副標題
description: "" // 描述
keywords:
author: Howie Chen // 作者
language: en
timezone: "Asia/Taipei" // 時區

# URL

## If your site is put in a subdirectory, set url as 'http://yoursite.com/child' and root as '/child/'
url: http://yoursite.com
root: /blog/  // 將此區改為github的位置

# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
type: git //使用方法
repo: git@github.com:howie12207/blog.git //SSH連結
branch: master //預設分支
