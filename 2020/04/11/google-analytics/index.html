在Hexo中載入Google-Analytics

發表於 2020-04-11 分類於 google-analytics

註冊 GA

先到 Google Analytics 註冊

取得 ID

在 管理 -> 追蹤資訊 -> 追蹤程式碼 -> 追蹤 ID

添加到 Hexo 中

themes/_config.yml 更改 ID

google_analytics:
  tracking_id: UA-163289150-1
