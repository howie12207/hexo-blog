for/in & for/of

Posted on 2020-04-13 In javascript

for…in 取得物件內的所有 key

let person = {name:'hulk', age:20};
for (x in person){
    console.log(x); // name, age
}

可再透過得到的 key 取得其值

for (x in person){
    console.log(person[x]); // hulk, 20
}

for…of 取得迭代型物件(陣列、Maps、Nodelists…等)，取出其值

let cars = ["asdf", "qw", "zz"];
for (x of cars) {
  console.log(x); // asdf, qw, zz
}
