cookies, local storage, session storage

Posted on 2020-04-13 In javascript

簡單比較下 cookies, local storage, session storage 三者差異

cookies

  • 儲存量: 4kb
  • 期限: 可透過手動設定期限
  • 取得範圍: 任一視窗、瀏覽器
  • 儲存位置: 瀏覽器及伺服器
  • 是否隨著 http req/res: 是

local storage

  • 儲存量: 三者中最大
  • 期限: 只有當使用者瀏覽器的快取/local stored data 才會消失
  • 取得範圍: 任一視窗、瀏覽器
  • 儲存位置: 瀏覽器
  • 是否隨著 http req/res: 否

session storage

  • 儲存量: 至少 5MB
  • 期限: 當分頁或瀏覽器被關閉，即消失
  • 取得範圍: 需同分頁
  • 儲存位置: 瀏覽器
  • 是否隨著 http req/res: 否
