Event queue

發表於 2020-04-13 分類於 javascript

以下列代碼作解說

function a() {
  console.log("a1");
  console.log("a2");
  console.log("a3");
  b();
  console.log("a4");
  setTimeout(() => {
    c();
  }, 0);
  console.log("a5");
}

function b() {
  console.log("b1");
  setTimeout(() => {
    c();
  }, 3000);
  console.log("b2");
  console.log("b3");
  c();
  console.log("b4");
  console.log("b5");
}

function c() {
  console.log("c1");
  console.log("c2");
  console.log("c3");
  console.log("c4");
  console.log("c5");
}

a();

JS 是逐行執行的，函數內遇到另一函數會先將另一函數執行完再回來，先進後出；遇到非同步的會放到 queue 佇列，待 stack 清空後再逐一執行，先進先出。
